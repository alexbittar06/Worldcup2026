# Worldcup2026

This project is a static World Cup simulator website.

## Features

- Choose the order of each team in every group (1-4)
- Generate the Round of 16 bracket automatically using the standard World Cup pairing format
- Pick winners through Quarterfinals, Semifinals, and the Final
- See the ultimate tournament winner displayed at the end

## Run locally

Open `index.html` in your browser.
